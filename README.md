## Installation

1. Clone the repository:
```bash
 git clone https://github.com/yourusername/yourproject.git
```

2. Install dependencies:
```bash
 npm install --legacy-peer-deps
```

3. Change the url to your API
```js
export const url = YOUR_URL
```
    
4. Run the project
```bash
 npm run android    # For android
 npm run ios        # For ios
 npm run web        # For browser
```

### Tests
#### Description
Provides various types of tests for users to complete.
#### Features
- **Multiple Sentence Tests**: Tests that involve multiple sentences.
- **Input Tests**: Tests that require user input.
- **Word Tests**: Vocabulary tests focusing on word recognition and usage.
- **Tests with Images**: Visual tests involving images.
- **Routes Tests**: Tests involving spatial or directional skills.
- **Multiple Choice Tests**: Traditional multiple-choice questions.
